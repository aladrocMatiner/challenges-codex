# Writeup - Attachment Flag (Beginner Friendly)

You’ve got this. This is a calm, no-pressure challenge that teaches careful inspection.

## Goal
Download the attached file and find the flag inside.

## Step 1: Download the File
Use the attachment link in the challenge and save the file locally.

## Step 2: Read the File
Open it with a text viewer or use the terminal:

```bash
cat evidence.txt
```

Look for something like `flag{...}`.

## Tip
If the file looks large, try searching for `flag{` in your editor.

## Recap
1. Download
2. Open
3. Find the flag
4. Submit

Great job — this is exactly how many real forensic tasks start.
